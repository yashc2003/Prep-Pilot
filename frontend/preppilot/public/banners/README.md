# Hero banner videos

Put your video banner files in this folder (MP4 recommended).

The landing page slider expects these paths:

- `public/banners/preppilot-1.mp4`
- `public/banners/preppilot-2.mp4`
- `public/banners/preppilot-3.mp4`

If the videos are missing, PrepPilot will automatically show the poster (image) fallback.
